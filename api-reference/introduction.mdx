## Authentication

L'ensemble des routes de notre API vous authentifient grace au **_token_** présent dans le champ **_'auth'_** du header.

<Note>
  Vous pouvez récuperer ce token via la route de **_login_**
</Note>

```json
"security": [
  {
    "auth": []
  }
]
```
